# Theme UI components

These components are not bundled with the boilerplate, they are optional dependencies for your design system and application

Find out about them all [here](https://theme-ui.com/components)

### Example

```jsx
<Box
  p={4}
  bg='highlight'>
  <Flex
    sx={{
      alignItems: 'center',
    }}>
    <Heading>
      Components
    </Heading>
    <Button ml='auto'>
      Beep
    </Button>
  </Flex>
</Box>
```

# Box

https://theme-ui.com/components/box

# Grid

https://theme-ui.com/components/grid
